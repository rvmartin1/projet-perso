Exemple de Serveur en Spring Boot qui expose des services REST

Issue de Spring Boot Tutorial

Présentation

Un serveur a été créé avec Spring Boot. Il est associé à une BDD composée d'une table Etudiant. Le serveur expose 3 services GET / POST / DELETE, qui ont les fonctionnalités suivantes :

- GET : récupérer la liste d'étudiants de la table ETUDIANT

- POST : ajouter un étudiant dans la table ETUDIANT

- DELETE : supprimer un étudiant de la table ETUDIANT

Technologies/Outils

Pour le back : Java, Spring Boot, REST, PostGre, Intellij

Pour le front (la page sur laquelle nous nous trouvons) : Javascript;, HTML, CSS, Visual Studio, Intellij, Tomcat
